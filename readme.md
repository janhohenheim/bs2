## Usage

- Download Counter-Strike 2 including its Workshop Tools from Steam. See [the setup guide](./docs/setup.md) for how to do that.
- Download the BS2 installer from the [releases page](https://github.com/janhohenheim/bs2/releases/latest) and run it
- Run BS2, and perform the one-time setup as indicated on screen
- (optional) Delete Counter-Strike 2, we don't need it anymore. See [the setup guide](./docs/setup.md) again.
- Programmers: see TODO for how to set up your Bevy project to work with BS2
- In BS2 create a new project in the "Projects" tab and launch the tools.
